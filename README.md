# To use this configuration:

## First of all:
use the `install` script to install all/most the apps/tools:
```
chmod +x install.sh
./install.sh
```

## Then :

### 1. Download `stow`:
```
sudo pacman -S stow
```

### 2. Apply the configuration for the desired one:
- if you wanted for zsh, do:
  ```
  stow zsh
  ```
